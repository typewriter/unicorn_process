    class UnicornProcess

    unicorn_railsの情報を収集するクラス

    Authors

    typewriter

    Version

    0.52

    License

    MIT

    動作条件

    • /proc ファイルシステムが利用できる

    • unicorn_rails によってRailsアプリケーションが動作している

    • netstatが利用できる

    Sample

    require './unicorn_process.rb'
processes = UnicornProcess.processes
processes.each { |process|
  puts "PID: #{process.pid}, ポート: #{process.port}, アプリディレクトリ: #{process.working_directory}"
}

    CACHED_RESULT

    動作状況を調べるためのシステムコールの実行結果をキャッシュします。 (複数の動作状況を調べたり、繰り返しメソッドを呼び出したりする場合にパフォーマンスが向上します) キャッシュをクリアするには、 clear クラスメソッドを呼び出します。

    processes() click to toggle source

    動作しているunicorn_railsを検索し、UnicornProcessの配列を返します。

    # File unicorn_process.rb, line 41
def self.processes
  unicorn_processes = []

  ps_result = %xps aux -ww`
  ps_result.each_line { |e|
    if e =~ %r^.+?\s+(\d+)\s+.+\s+unicorn_rails master/
      process_id = $1.to_i
      unicorn_processes << UnicornProcess.new(process_id) if e !~ %r\(old\)/
    end
  }

  unicorn_processes
end

    config_path() click to toggle source

    unicorn_rails 起動時に -c または –config-file で与えた設定ファイルへのパスを示します。 指定されていない場合は nil です。 また、作業ディレクトリが取得できない場合は相対パスを示します。

    # File unicorn_process.rb, line 140
def config_path
  cmdline = File.read("#{proc_path}/cmdline")
  if cmdline =~ %r(?:\-c|\-\-config\-file)(?:\s+|=)([^ ]+)/
    if working_directory
      File.expand_path($1.unpack("Z*").first, working_directory)
    else
      $1.unpack("Z*").first
    end
  else
    nil
  end
end

def path
  cmdline = File.read("#{proc_path}/cmdline")
  if cmdline =~ %r\-\-path(?:\s+|=)([^ ]+)/
    $1.unpack("Z*").first
  else
    "/"
  end
end

    port() click to toggle source

    unicorn_railsが接続を受け付けるポートを示します。 戻り値は配列です。配列の各要素は :port, :type キーを持ったハッシュです。:typeはソケットの種類(“unix”: Unixドメインソケット, “tcp”: TCPソケット)、:portはポート番号(I-Node番号)を表します。 権限等の問題で取得できない場合があります。

    # File unicorn_process.rb, line 63
def port
  # まあ、コンフィグを解釈するとかめんどいわけで。
  ports = []

  netstat_result = (CACHED_RESULT ? (defined?(@@netstat) ? @@netstat : @@netstat = %xnetstat -anp`) : %xnetstat -anp`)
  childs = child_pids
  netstat_result.each_line { |e|
    if e =~ %r\s+#{@pid}\// || (!childs.empty? && e =~ %r\s+(?:#{childs.join('|')})\//)
      # unix domain
      if e =~ %rLISTENING\s+(\d+)\s+/
        ports << { :port => $1.to_i, :type => "unix" }
      elsif e =~ %r[^:]\:(\d+)\s+.+?\s+LISTEN/
        ports << { :port => $1.to_i, :type => "tcp" }
      end
    end
  }
  ports
end
    ruby_version() click to toggle source

    動作している ruby インタプリタのバージョンを示します。 (rvmなどを利用している場合向けのメソッドです) 権限等の問題で取得できない場合は nil です。

    # File unicorn_process.rb, line 111
def ruby_version
  begin
    exe = File.readlink("#{proc_path}/exe")

    if CACHED_RESULT
      @@rubies = {} if !defined?(@@rubies)
      (@@rubies[exe] ? @@rubies[exe] : @@rubies[exe] = %x#{exe} -v`.chomp)
    else
      %x#{exe} -v`.chomp
    end
  rescue
    nil
  end
end
    worker() click to toggle source

    現在の worker の数を示します。

    # File unicorn_process.rb, line 95
def worker
  workers = 0

  ps_result = (CACHED_RESULT ? (defined?(@@psaxl) ? @@psaxl : @@psaxl = %xps axl -ww`) : %xps axl -ww`)
  ps_result.each_line { |e|
    if e =~ %r^\d+\s+\d+\s+\d+\s+#{@pid}\s+.+\s+unicorn_rails worker/
      workers += 1
    end
  }

  workers
end
